# 📈 AI Research Growth Across Major ML Conferences

> Analysis of **40,000+ research papers** published in **NeurIPS, ICML, and ICLR (1987–2024)** using Python, data analysis, and unsupervised learning techniques.

---

## 🎯 Overview

This project analyzes the growth of AI research by studying publication trends across three leading machine learning conferences.

The analysis focuses on understanding:

- 🌍 Global AI research growth
- 📊 Country-wise publication trends
- 🏛️ Research concentration across countries
- 📈 Long-term publication patterns
- 🔬 Emerging research hubs using clustering techniques

This project was completed as part of the **Introduction to Data Science (CS328)** course at **IIT Gandhinagar**.

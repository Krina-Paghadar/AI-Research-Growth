# AI Research Growth Across Major ML Conferences

Analysis of research trends across **NeurIPS**, **ICML**, and **ICLR** (1987–2024) using Python and data analysis techniques.

---

## Overview

This project analyzes more than **40,000 research papers** published in NeurIPS, ICML, and ICLR to study the global growth of AI research.

The analysis explores publication trends, country-wise contributions, research concentration, and emerging research hubs using data processing and clustering techniques.

---

## Dataset

- NeurIPS
- ICML
- ICLR
- Years: **1987–2024**

---

## Methods

- Data Cleaning
- Pandas
- Country Extraction
- CAGR Analysis
- Correlation Analysis
- KMeans Clustering
- Principal Component Analysis (PCA)

---

## Key Results

- 📄 Analyzed **40,000+ papers**
- 📈 Global CAGR: **14.46%**
- 🌍 Top-5 countries contributed **82.3%** of publications
- 📊 Year-output correlation: **0.662**
- 📉 Gini coefficient: **0.79**
